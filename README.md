# 🎬 CineSuggest API - Spring Boot

API para sugestões de filmes, desenvolvida em **Java 21** com **Spring Boot**, utilizando **Gradle** para build e **Docker** para orquestração.  
Banco de dados: **Oracle DB**.

---

## 🚀 Tecnologias utilizadas
- **Java JDK 21**
- **Spring Boot**
- **Gradle**
- **Docker**
- **Oracle Database**

---

## 📦 Como rodar o projeto

Certifique-se de ter o **Docker** instalado e rodando na máquina.  
Para subir os containers, execute:

```bash
docker-compose up
